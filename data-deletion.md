### üá¨üáß User Data Deletion

The **Trayomo** application does not use user accounts and does not store user data on external servers.

All user data, including:
- financial records (expenses);
- expense categories;
- data obtained through local notification text parsing,

is stored **locally on the user‚Äôs device**.

### Data deletion options

Users can manage and delete their data in the following ways:

1. **Partial data deletion**  
   The **Trayomo** app provides **delete controls for all data entities** (such as expenses, categories, and other records), allowing users to remove individual items selectively.

2. **Complete deletion of all application data**  
   Users can:
   - uninstall the **Trayomo** application from their device;  
   - or clear the application data via Android system settings.

After complete deletion, all user data is **permanently removed** from the device.

The **Trayomo** application does not retain backup copies of user data, and deleted data cannot be restored.
